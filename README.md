## mac编译unpv13e

下载源码：
```bash
git clone https://github.com/ltqthu/unpv13e-notes.git
```

进入路径：
```bash
cd unpv13e-notes/src
```
依次执行：
```bash
./configure
cd lib
make
cd ../libfree
make
cd ../libroute
make
cd ../intro
make daytimetcpcli
./daytimetcpcli 127.0.0.1
```
